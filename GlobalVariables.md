All global variables declared in MEL files from the `maya/scripts` directory, parsed and sorted so that you won't have to.

### Why?

These are scattered throughout the interwebs with no one spot to find them all. Some of these are useful to accessing UI elements like main window or timeline.

```py
from maya import mel
from PySide2 import QtWidgets
import shiboken2

mel_timeline = mel.eval("$tmpVar=$gPlayBackSlider")
ptr_timeline = MQtUtil.findControl(mel_timeline)
qt_timeline = shiboken2.wrapInstance(long(ptr_timeline), QtWidgets.QWidget)  # Exclude long() for Python 3
qt_timeline.parent().parent().setStyleSheet("border: 5px solid red;")
```

<br>

### How do I regenerate these?

Just run this.

```py
import re
import os

root = r"C:\Program Files\Autodesk\Maya2020\scripts"
matches = set()

for base, dirs, files in os.walk(root):
    for fname in files:
        if not fname.endswith(".mel"):
            continue

        path = os.path.join(base, fname)

        with open(path) as f:
            text = f.read()
            for match in re.finditer(r"\$g+[A-Z][A-Za-z]*", text):
                matches.add(text[match.start():match.end()])

print("\n".join(sorted(matches)))
```
